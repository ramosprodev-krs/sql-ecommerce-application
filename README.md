Project Start Date: January 15, 2026

Completion Date: February 10, 2026

Este projeto foi desenvolvido em Java utilizando o ecossistema Spring Boot 3.4.

Uma aplicação robusta de E-commerce focada em segurança, persistência de dados e integração com serviços de nuvem (AWS).
Tecnologias utilizadas:

    Java 21 (Spring Framework)

    Spring Security & JWT: Autenticação e autorização stateless.

    Spring Data JPA: Abstração de banco de dados e persistência.

    PostgreSQL: Banco de dados relacional para alta integridade de dados.

    AWS SES (Simple Email Service): Configuração de envio automático de e-mails de boas-vindas.

    Swagger (OpenAPI 3): Documentação interativa e testes de endpoints.

    Docker: Containerização do banco de dados e ambiente.

    Lombok: Produtividade e redução de código boilerplate.

    Insomnia: Validação e teste das rotas da API.
