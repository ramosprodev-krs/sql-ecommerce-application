This project was developed in **Java** using the **Spring Boot**.
A robust **E-commerce** application focused on security, data persistence, and cloud service integration (**AWS**).  

### Technologies used:    
- Java 21 (Spring Framework)  
- Spring Security & JWT  
- Spring Data JPA  
- PostgreSQL  
- AWS SES   
- Swagger  
- Docker  
- Lombok
